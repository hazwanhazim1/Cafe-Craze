# Café Craze ☕

A web-based 3D business simulation game built for my university degree project. In this tycoon-style game, players build, manage, and upgrade their very own bustling coffee shop. 

## 🎮 What is this game?

Café Craze is a prototype that focuses on resource management and 3D web graphics. As the player, you have to serve customers, manage your daily cafe operations, and make strategic choices to grow your business. 

I designed the complete game logic, user interface, and 3D scene rendering from scratch using standard web technologies.

## 🚀 Game Features

* **3D Environment:** A custom-rendered coffee shop scene built directly for the web browser.
* **Tycoon Mechanics:** Systems for tracking money, customer happiness, and cafe upgrades.
* **Interactive UI:** A responsive dashboard built with HTML and CSS to control the game state.
* **Game Design Document:** A full `Web Game Development (GDD).pdf` is included in this repository, showing my planning process from concept to the final prototype.

## 🛠️ Tools & Languages Used

* **Logic & 3D Rendering:** JavaScript (Canvas API / WebGL)
* **Interface:** HTML & CSS

## 🕹️ How to Play

Since this is a web-based game, no complex installation is required! 
1. Download the repository files to your computer.
2. Open the `index.html` file in any modern web browser (like Chrome or Edge) to start playing.
*(Note: You can also play the live version via the link provided in the repository details!)*
